kOS Bot Commands

  • Valorant Wiki:

    /{agent} /{weapon} /{location}

  • Valorant Username Db:

    /whois user {user}: {game}

  • Server Invite:

    /invite

  • Google Search:

    !google {search}

  • YouTube Search:

    !yt {search}

  • Twitch Search:

    !twitch {search}

  • Duel random user:

    /duel

  • Magic 8Ball:

    .8ball {question}

  • Take Note:

    !setnote {yournote} !viewnote !clearnote

  • Bot Latency:

    /ping

  • Server Event:

    /event {title} {date} {time}

  • Server Announcement:

    /announce {message}

  • Bot Command List:

    /help /cmd
